# bamboo

Bamboo is a lightweight BAM file viewer written in [Crystal](https://crystal-lang.org/).  
It provides a simple native GUI for browsing alignment records.

## Installation

```
git clone https://github.com/bio-cr/bamboo
cd bamboo
shards build
```

## Usage

```
bin/bamboo
```
